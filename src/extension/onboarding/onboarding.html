Welcome to Link Scoop

Clean link extraction for the active tab

Link Scoop extracts directly from the toolbar icon, preserves page order by default, and adds a right-click menu so extraction is easy to find.

Quick start

  1. Look for the bright Link Scoop icon in the Firefox toolbar. Firefox places it in the main toolbar by default.
  2. If Firefox moved it into the Extensions button, open that menu and choose Pin to Toolbar.
  3. Click the icon to extract links right away, then use the results page to adjust sort order and filters.
  4. Right-click any webpage and choose Link Scoop → Extract Links for a faster workflow.
  5. Use Ctrl+Shift+L on Windows/Linux or Command+Shift+L on macOS to extract links from the active tab without using the toolbar.

Local file support

To scan a local file:/// HTML page, Link Scoop may need permission to read local files.

  1. Open about:addons.
  2. Select Link Scoop.
  3. Open the permissions section and enable local file access if Firefox asks for it.
  4. Return to the file tab and run Link Scoop again.

Note: Saved HTML pages may not contain all links from the live site. If the original page used JavaScript to load content dynamically, those links won't exist in the saved file. Links that were relative on the live site (like /news or /products) will appear as file:/// paths instead of full URLs. This is expected behavior, not a bug.
